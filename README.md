Данный проект был создан в целях демонстрации моих навыков программировании с помощью библиотеки React.
Я создала список привычек, в котором хранятся данные о них: название, цвет привычки, описание и отмеченные дни.

# ИНСТРУКЦИЯ ДЛЯ ЗАПУСКА

1. Установить зависимости `npm i`
2. Запустить приложение `npm run dev`
